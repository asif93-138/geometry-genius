1. What is DOM and what is the purpose of DOM?
- DOM means Document Object Model. It treats XML or HTML document as tree structure and is used by JavaScript to maipulate HTML element.
2. How will you select HTML elements using DOM? Name the DOM methods.
- Using document.getElementsByTagName(), document.getElementsByClassName(), document.getElementById() etc.
3. What is event bubble?
- Event bubble is running a JS event in HTML elements in upward or downward way.
4. What is a callback function and why will you use it?
- It's a kind of recursive function. It calls another function within a function.
